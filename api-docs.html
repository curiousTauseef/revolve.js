Methods

revolve.gauge( context[, options])

Render a themed gauge to the <canvas> element specified by the context parameter. Optionally, include an options object for the 2nd parameter.

Returns: An object of type RadialGauge.

revolve.clock( context[, options])

Render a themed clock to the <canvas> element specified by the context parameter. Optionally, include an options object for the 2nd parameter.

Returns: An object of type AnalogClock.

Properties

revolve.themes

An object collection of all the predefined themes. Only available in packaged builds (revolve.pkgd.js and revolve.pkgd.min.js).

Options

Both the gauge and clock methods accept an options object for the second parameter.

opts.center

Type: Array (optional)

The [x,y] coordinate of the center of the clock or gauge. If omitted, defaults to the canvas midpoint.

let opts = {
  center: [100,100]
};
Revolve.gauge( ctx, opts );

opts.layout

Type: String (optional)

Describes the layouting behavior of the clock or gauge as the <canvas> element is resized. Valid values are:

  • auto: The clock is centered and resized to the maximum extent without overflowing the canvas.
  • static: The size of the clock/gauge stays constant when the <canvas> is resized.
  • scale: The size of the clock scales with the size of the canvas.
let opts = {
  layout: 'contain'
};
Revolve.gauge( ctx, opts );

opts.mode

Type: String (optional)

[Clocks only.] The animation style of the clock. Can be either continuous or discrete. Default is discrete.

let opts = {
  mode: 'continuous'
};
Revolve.clock( ctx, opts );

opts.radius

Type: Number (optional)

The initial size of the clock or gauge. If omitted, defaults to Math.min(w,h) / 2, where w and h are the width and height of the <canvas> element.

let opts = {
  radius: 256
};
Revolve.gauge( ctx, opts );

opts.theme

Type: String or Object (optional)

The name of a predefined Revolve.js theme OR a JavaScript/JSON theme object.

let opts = {
  theme: 'unitless'
};
Revolve.gauge( ctx, opts );

opts.value

Type: Number or String (optional)

The initial value of the clock or gauge.

let opts = {
  value: 100
};
Revolve.gauge( ctx, opts );
